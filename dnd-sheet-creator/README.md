# Dungeons & Dragons Character Sheet Creator

This project is a web-based application that allows users to create and manage character sheets for Dungeons & Dragons. The application provides a user-friendly interface for entering character details, viewing saved characters, and managing character attributes.

## Project Structure

```
dnd-sheet-creator
├── src
│   ├── index.html        # Main entry point for the application
│   ├── create.html       # Character creation page
│   ├── list.html         # Character list page
│   ├── css
│   │   └── styles.css     # CSS styles for the application
│   └── js
│       ├── app.js        # Main application logic
│       ├── ui.js         # User interface interactions
│       └── storage.js     # Data storage management
├── .gitignore            # Files and directories to ignore by Git
└── README.md             # Project documentation
```

## Features

- Create new character sheets with customizable attributes.
- View a list of saved characters.
- Edit and delete character sheets.
- Responsive design for various screen sizes.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd dnd-sheet-creator
   ```

3. Open `src/index.html` in your web browser to start using the application.

## Usage

- Use the "Criar Ficha" button to create a new character.
- Fill out the character details in the form and save.
- Use the "Suas Fichas" button to view all saved characters.
- Click on a character to view its details or delete it.
